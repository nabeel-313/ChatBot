# ChatBot

## Clone repro
```
https://github.com/nabeel-313/ChatBot.git
```

## create venv and activate
```
conda create -p chatbot python==3.12 -y
conda activate ./chat
```

## Install Requiments
```
pip install -r requirements.txt
```

## Run
```
python app.py
```
